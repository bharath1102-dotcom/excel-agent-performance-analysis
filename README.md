# Agent Performance Analysis Dashboard using Excel

##  Project Overview

This project presents an agent performance dashboard built using Microsoft Excel to evaluate customer support efficiency. The dashboard analyzes key metrics such as performance scores, SLA, CSAT, and reopen rates to identify high-performing agents and areas for improvement.

---

## Tools & Techniques Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Data Cleaning & Transformation
* Slicers and KPI Cards
* Data Visualization

---

## Dataset Description

The dataset includes:

* Agent names
* Performance scores
* SLA (Service Level Agreement) metrics
* CSAT (Customer Satisfaction) scores
* Reopen rates
* Process categories (Billing, Sales, Tech Support)

---

##  Dashboard Features

###  Key Performance Indicators (KPIs)

* Average Performance Score: 74%
* Star Performers Count: 2
* Needs Coaching Count: 1
* Average SLA Score: 63%

---

###  Agent Performance Ranking

* Ranks agents based on performance scores
* Identifies top performers and low performers

---

###  SLA vs CSAT Analysis

* Shows relationship between service quality (SLA) and customer satisfaction (CSAT)
* Helps understand service effectiveness

---

###  Reopen Rate by Agent

* Measures how often issues are reopened
* Identifies potential quality issues in resolution

---

###  Interactive Filters (Slicers)

* Agent Name
* Process (Billing, Sales, Tech Support)
* Performance Category

---

##  Key Insights
